This is a project written in C- OpenCV Lib.

Features of Program:

1) Video captured from PC Camera and Display of the video
2) RGB values are computed from each of the frame
3) Histogram on a scale 1-256 for Red, Green and Blue values
4) All three values of the histogram is displayed along with the video display.
5) This project can be extended any dual vision requirement

Requirements:

a) Code Blocks, Visual Studio, or Dev C++
b) OpenCV installed in any of the three.
